# Repository Instructions

Before committing or pushing, check the time in America/New_York. From 07:42 through 17:36, Monday through Friday, work only on uncommitted changes or local-only branches. Outside that window, squash the final diff onto a new publish/* branch from origin/main, commit it, push it, and open a pull request. Never publish commits created during the protected window, push local-only branches, or push directly to main.
